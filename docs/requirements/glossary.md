# Glossary

| Thuật ngữ | Ý nghĩa |
|---|---|
| Employee | Nhân viên trong hệ thống |
| EmployeeContract | Hợp đồng lao động của nhân viên |
| ContractCode | Mã hợp đồng lao động |
| ContractType | Loại hợp đồng lao động |
| BaseSalary | Lương cơ bản ghi nhận trên hợp đồng |
| SalaryHistory | Lịch sử lương theo thời điểm |
| SocialInsuranceHistory | Lịch sử mức đóng BHXH theo thời điểm |
| Active Contract | Hợp đồng đang hiệu lực |
| Stakeholder | Người liên quan có quyền cung cấp/chốt yêu cầu |
| Source of Truth | Nguồn thông tin chính, được ưu tiên khi có mâu thuẫn |
